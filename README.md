## Current Focus

- **Integrated Photonics:** Silicon photonics, waveguide engineering, and mode-division multiplexing systems.
- **Nonlinear Quantum Photonics:** SPDC-oriented architectures and quasi-BIC photonic concepts.
- **Physics-Informed Design:** PINN-assisted inverse design and multiphysics optimization.
